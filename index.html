Rock, Paper, Scissors Game

Developed an interactive Rock, Paper, Scissors game using vanilla JavaScript, focusing on dynamic UI and gameplay logic. The game features user and computer interactions with real-time score tracking and visual feedback for wins, losses, and draws. The layout was styled using responsive design principles with CSS Flexbox for a seamless user experience. Integrated hover effects and dynamic message updates to enhance player engagement.

Tic-Tac-Toe Game

Developed an interactive Tic-Tac-Toe game using JavaScript, designed with a responsive layout and smooth user interactions. Implemented gameplay logic for two players, with alternating turns between "X" and "O". The game detects winning combinations or draws using a predefined algorithm. The interface was styled using CSS, incorporating flexbox for layout and button states for interactivity. Features include a reset function and a new game option, enhancing the user experience.
